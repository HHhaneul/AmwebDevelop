배열
1. 배열이란?

- 자료를 순차적으로 관리하는 구조
- 같은 자료형(공건의 크기 동일)
- 물리적인 나열 구조 (순차 자료 구조)
	-> 공간의 위치 계산이 쉽다.
	-> [ ]: 인덱스 연산자, 0, 1, 2, ... : 공간의 위치 연산


- 자료를 순차적으로 관리하는 구조

2. 배열 선언과 초기화
	자료형[] 배열명 = new 자료형[공간의 개수];
	자료형 배열명[] = new 자료형[공간의 개수];


3. 배열 사용하기
	- 인덱스 연산자
		[]: 배열 공간의 위치를 계산해 주는 연산자
		
		참고)
			인덱스: 0부터 시작하는 순서

	- 선언과 동시에 초기화
	자료형[] 배열명 = new int[] { 값1, 값2, 값3, ... , 값n }
	자료형[] 배열명 = { 값1, 값2, 값3, ... , 값n }

참고)
	기본 자료형: 초기값
		정수형 자료형(byte, short, int, long) -> 0
		실수형 자료형(float, double) -> 0.0
		물리형 자료형(boolean) -> false
	
	참조 자료형
		예) String - 초기값 null
